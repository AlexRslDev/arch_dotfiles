# File Explorers

### Nautilus

```
sudo pacman -S --needed nautilus gvfs file-roller sushi
```

![Nautilus](https://static.gnome.org/catalog/app-screenshot/org.gnome.Nautilus/thumbnails.png)

### Dolphin

```
sudo pacman -S --needed dolphin ark ffmpegthumbs kdegraphics-thumbnailers
```

![Dolphin](https://upload.wikimedia.org/wikipedia/commons/d/db/Dolphin_%28file_manager%29_24.12.0_screenshot.webp)

### Thunar

```
sudo pacman -S --needed thunar gvfs thunar-volman thunar-archive-plugin tumbler
```

![Thunar](https://opensource.com/sites/default/files/2022-09/thunarfilemanager.png)

### Yazi

```
sudo pacman -S --needed yazi ffmpegthumbnailer imagemagick poppler fd ripgrep
```
